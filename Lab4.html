# Function to draw a 3D-like fractal tree using recursion
def fractal_tree_3d(length, depth, angle):
    if depth == 0:
        return
    else:
        forward(length)
        left(angle)
        fractal_tree_3d(length * 0.7, depth - 1, angle)  # Left branch
        right(2 * angle)
        fractal_tree_3d(length * 0.7, depth - 1, angle) # Right branch

        left(angle)
        backward(length)

        # Simulate depth by slightly shifting the branches
        if depth > 2: # Avoid shifting too early
          penup()
          right(angle/2)
          forward(depth * 2)
          pendown()
          left(angle/2)
          fractal_tree_3d(length * 0.5, depth - 2, angle) # depth branch
          penup()
          backward(depth * 2)
          left(angle/2)
          pendown()
          right(angle/2)


# Initialize the turtle
initializeTurtle(initial_window_size=(800, 800))

# Set up the turtle's starting position and orientation

length = 80  # Initial length of the main trunk
depth = 7     # Recursion depth (number of branching levels)
angle = 25 # branching angle
# Draw the 3D-like fractal tree
fractal_tree_3d(length, depth, angle)

# Display the drawing

Conclusion

Explanation of the code:

  • midpoint(): This function calculates the midpoint of a triangle by averaging the coordinates of its three vertices.
  • sierpinski_3d(): This recursive function generates the 3D Sierpiński triangle. At each recursion depth, it subdivides the triangle into smaller triangles, removing the central portion to create the fractal structure.
  • p1, p2, p3: These represent the coordinates of the three corners of the initial equilateral triangle from which the fractal starts.
  • depth: This parameter controls the recursion depth, determining how many iterations of subdivision will occur. A higher depth results in a more detailed fractal structure.
  • Poly3DCollection: This is a feature from the Matplotlib library used to visualize the 3D Sierpiński triangles by rendering the smaller triangles as 3D polygons.
