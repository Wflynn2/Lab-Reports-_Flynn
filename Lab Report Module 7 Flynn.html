Introduction

Introduction to the Mandelbrot and Julia Sets

The Mandelbrot and Julia sets are two of the most impressive examples of mathematical fractals. They are rooted in complex numbers and iterative functions. The images that are produced are infinte and self similarity. The function is used is f(z)=z^2+c. The z and c are complex numbers. The behavior of this function under repeated iteration, is whether the sequence goes off to infinity or remains bounded, determines whether a point belongs to the Julia set or to the Mandelbrot set.

The Julia set is generated by fixing a constant c and observing how different starting valuse of z behave under iteration. Depending on the choice of c, the resulting Julia set can be connected and smooth or fragmented and dust-like.

Both the Mandelbrot and the Julia set has fascinated mathematicians and artists for many years. Exploring them opens a window into the richness of patterns hidden in the intricate and fabric of mathematics.

Mandelbrot Set Function Summary

def mandelbrot(c, max_iter): z = 0 for i in range(max_iter): z = z*z + c if abs(z) > 2: return i return max_iter

- Purpose: Checks whether a complex number c belongs to the Mandelbrot set. - z: Starts at 0 and is updated as z = z² + c. - Escape condition: If |z| > 2, the point is considered to escape. - Returns: The number of iterations before escape (or max_iter if it doesn't escape).

Julia Set Function Summary

def julia(z, c, max_iter): for i in range(max_iter): z = z*z + c if abs(z) > 2: return i return max_iter

Purpose: Checks whether a complex number z belongs to the Julia set for a given constant c. z: Starts at a chosen point in the complex plane and iterates using z = z² + c. c: Constant value that defines the specific Julia set. Returns: The number of iterations before z escapes beyond 2.

import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation
from numba import jit

# JIT-optimized Mandelbrot function
@jit(nopython=True)
def mandelbrot(c, max_iter):
    z = 0
    n = 0
    while abs(z) <= 2 and n < max_iter:
        z = z*z + c
        n += 1
    return n

@jit(nopython=True)
def mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter):
    r1 = np.linspace(xmin, xmax, width)
    r2 = np.linspace(ymin, ymax, height)
    n3 = np.empty((width, height))
    for i in range(width):
        for j in range(height):
            n3[i, j] = mandelbrot(r1[i] + 1j * r2[j], max_iter)
    return n3

Conclusion

The Mandelbrot and Julia sets are two of the most iconic and visually striking examples of fractals in mathematics and computer graphics. Though both are generated using the simple iterative formula they reveal incredibly complex and infinitely detailed structures when visualized.

The Mandelbrot set serves as a map or catalog of Julia sets—each point in the Mandelbrot set corresponds to a different type of Julia set, ranging from connected to completely fragmented forms.
