Introduction

Overview This Python script generates and visualizes the Mandelbrot Set — a famous fractal defined in the complex plane. The Mandelbrot Set is composed of complex numbers for which the function f(z) = z² + c does not diverge when iterated from z = 0. The visualization is created using NumPy for numerical operations, Matplotlib for plotting, and Numba's JIT compiler to optimize performance.

Function Documentation @jit(nopython=True) Numba's @jit decorator is used to speed up numerical computations by compiling the Python functions to machine code at runtime. nopython=True enforces that the function does not fall back to slower Python mode, which boosts performance significantly.

mandelbrot(c: complex, max_iter: int) -> int Purpose: Determines the number of iterations it takes for the complex sequence zₙ₊₁ = zₙ² + c (starting from z = 0) to exceed an absolute value of 2. This helps to determine whether a complex number c is in the Mandelbrot Set.

Parameters: c: A complex number to test. max_iter: Maximum number of iterations before deciding the point is in the set.

Returns: The number of iterations before the sequence escapes, or max_iter if it doesn't. mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter) -> np.ndarray

Purpose: Generates a 2D array representing the Mandelbrot Set over a specified portion of the complex plane.

Parameters: xmin, xmax: Range of the real axis. ymin, ymax: Range of the imaginary axis. width, height: Resolution of the output image. max_iter: Maximum number of iterations per point.

Returns: A 2D NumPy array where each element represents the number of iterations for the corresponding complex point.

How It Works: Constructs linearly spaced arrays for the real (r1) and imaginary (r2) parts of complex numbers. Loops through every point in the 2D space, evaluates its membership in the Mandelbrot Set using the mandelbrot() function, and stores the result in an output array.

import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation
from numba import jit

# JIT-optimized Mandelbrot function
@jit(nopython=True)
def mandelbrot(c, max_iter):
    z = 0
    n = 0
    while abs(z) <= 2 and n < max_iter:
        z = z*z + c
        n += 1
    return n

@jit(nopython=True)
def mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter):
    r1 = np.linspace(xmin, xmax, width)
    r2 = np.linspace(ymin, ymax, height)
    n3 = np.empty((width, height))
    for i in range(width):
        for j in range(height):
            n3[i, j] = mandelbrot(r1[i] + 1j * r2[j], max_iter)
    return n3

center_x, center_y = -0.743643887037151, 0.131825904205330  # Deep zoom region
zoom_factor = 0.95  # How much to zoom in each frame
num_frames = 30     # Number of frames in animation
width, height = 800, 800
max_iter = 500
cmap_choice = 'twilight_shifted'  # Try others: 'hot', 'inferno', 'plasma', etc.

# =======================
# ANIMATION SETUP
# =======================

fig, ax = plt.subplots(figsize=(8, 8))

def animate(i):
    ax.clear()
    zoom = zoom_factor ** i
    range_x = 0.01 * zoom
    range_y = 0.01 * zoom
    xmin = center_x - range_x
    xmax = center_x + range_x
    ymin = center_y - range_y
    ymax = center_y + range_y

    mandelbrot_img = mandelbrot_set(xmin, xmax, ymin, ymax, width, height, max_iter)
    ax.imshow(mandelbrot_img.T, extent=[xmin, xmax, ymin, ymax], cmap=cmap_choice)
    ax.set_title(f"Mandelbrot Zoom Frame {i+1}")
    ax.axis('off')

ani = FuncAnimation(fig, animate, frames=num_frames, interval=150)
ani.save('mandelbrot_zoom.gif', writer='pillow')  # Save as GIF

plt.show()
